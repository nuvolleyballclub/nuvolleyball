# AI Tool Disclosure — WCS 240 Final Project

Per the WCS 240 syllabus AI policy, this document logs any use of AI-powered tools in the creation of the NU Volleyball Club website and promotional video.

## Tool usage

### Claude (Anthropic)
**Used for:** initial scaffolding of the Next.js project structure, drafting placeholder copy for the About page, generating the Tailwind design system, writing README.md training instructions for the client. Later: a content rewrite pass that replaced the initial placeholder copy with facts sourced verbatim from the client's authoritative Tilda site (https://volleyballclubnu.tilda.ws/).

**Extent:** Code structure, component scaffolding, and first-draft copy were generated by Claude and then reviewed, edited, and personalized by the group. The later rewrite pass mapped the Tilda-sourced facts (mission, departments, leadership, presidents, coaches, rosters, events, sponsors) into the codebase; the underlying facts are the client's, the wording around them was assisted by Claude.
